# Time Series - Watch Website

## Project Description

Time Series is a modern front-end watch store website designed to showcase luxury and premium watches. The website features a clean and responsive user interface with multiple pages, allowing users to browse products, learn about the brand, read blogs, contact the company, and view a shopping cart layout.

This project was developed using HTML5 and CSS3 as part of a AICT project in  my first semester.

---

## Features

- Responsive website layout
- Attractive homepage with banner/video
- Product showcase with watch collections
- About Us page
- Blog page
- Contact page with form
- Shopping cart page
- Navigation bar
- Image hover effects
- Modern dark theme
- Responsive design
